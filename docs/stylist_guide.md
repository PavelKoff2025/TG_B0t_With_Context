# Что надеть сегодня — мини-гайд стилиста

Бытовой помощник: по погоде и планам получаешь **3 готовых образа** и короткие заметки.

Этот гайд собран по итоговому ответу модели (прогон 2, после улучшения промпта).

---

## Исходные данные

- **Город / погода:** Москва, +5°C, дождь, сильный ветер  
- **Планы:** офис, вечером короткая прогулка до метро  
- **Модель:** gpt-4o-mini · temperature 0.3  

---

## Что делать по шагам

### 1. Выбери один из трёх образов

| № | Верх | Низ | Обувь |
|---|------|-----|-------|
| 1 | Тёплый свитер | Тёмные джинсы | Водоотталкивающие ботинки |
| 2 | Удлинённый плащ | Брюки-чиносы | Утеплённые кроссовки |
| 3 | Тонкая куртка | Юбка-карандаш | Резиновые сапоги |

**Рекомендация на этот день:** образ **№1** — теплее и уместнее для офиса при ветре.

### 2. Перед выходом проверь notes

1. Возьми **зонт** и **тёплый шарф**  
2. **Не бери** лёгкие ткани — не защитят от ветра и дождя  

### 3. Вечером (прогулка до метро)

- Оставь водозащитную обувь  
- Застегни верх / подними воротник от ветра  
- Зонт держи под рукой

## Как получить новый совет самому

```bash
cd ~/Desktop/TG_Bot_With_Contecst
source venv/bin/activate
python openai_direct.py
```

1. Выбери промпт **11** (Мини-стилист по погоде)  
2. Temperature: **0.3**  
3. Введи свою погоду и планы  
4. Получи JSON → можно снова сохранить в `docs/`  

Или покажи сохранённый артефакт:

```bash
python stylist_show.py
```
